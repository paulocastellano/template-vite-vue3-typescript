# Template for Front End applications

## This template contains:

- Vite 2
- Vue 3
- Typescript
- Tailwind CSS

```sh
npm install
# or
yarn add
```

### start the project

```sh
npm run dev
```

### build the project

```sh
npm run build
```

### husky permissions

```sh
chmod +x .husky/_
chmod +x commit-msg
chmod +x pre-commit
chmod +x pre-push
```
